# Resume App - React Native (Web + Mobile)

A cross-platform resume/portfolio application built with React Native, supporting Web, iOS, and Android platforms.

## Quick Start

```bash
# Install dependencies
yarn install

# Run on web
yarn web

# Run on iOS
yarn ios

# Run on Android
yarn android
```

## Tech Stack

- React Native
- React Native Web
- NativeWind (Tailwind CSS)
- React Native Reanimated
- Webpack (for web builds)
- Metro Bundler (for mobile builds)

See [README_RN_SETUP.md](./README_RN_SETUP.md) for detailed setup instructions.
